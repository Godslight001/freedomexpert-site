# Freedom Expert Website Rebuild

Professional multi-page static website with concise navigation and CMS-ready page templates.

## Pages

- `index.html` - Home
- `programs/index.html` - Programs listing
- `programs/financial-freedom.html`
- `programs/leadership-lab.html`
- `programs/family-stewardship.html`
- `resources/index.html` - Resources listing
- `resources/blog.html` - Blog listing
- `resources/article.html` - Blog article template (CMS-ready)
- `resources/podcast.html` - Podcast listing
- `resources/episode.html` - Podcast episode template (CMS-ready)
- `resources/books.html`
- `resources/downloads.html`
- `about/index.html`
- `contact/index.html`

## Assets

- `assets/css/styles.css` - Brand design system
- `assets/js/main.js` - Nav + animation behavior
- `assets/images/freedom-expert-logo.svg` - Logo placeholder

## Logo

Replace `assets/images/freedom-expert-logo.svg` with your final logo asset (same filename) to apply it site-wide.

## Run locally

```powershell
python -m http.server 8080
```

Open `http://localhost:8080`.
